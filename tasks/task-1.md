# Task - 1 [JavaScript - Control Structures]

Code : **Handwritten**

Output : **Printed**

Papersize : **A4**

## Question 1

Write a JavaScript program that prints numbers from **1** to **20** [ using ``for`` loop ]:

- **Skip** numbers that are divisible by **3** using ``continue``.
- **Stop** the loop completely when the number reaches **15** using ``break``.
- For all other numbers, print the number.

## Question 2

Write a JavaScript program that checks numbers from **1** to **30** [ using ``for`` loop ]:

- If a number is even, print ``"Even: <number>"``.
- If a number is odd, print ``"Odd: <number>"``.
- **Skip** the number **17** using ``continue``.
- **Stop** the loop when the number **25** is reached using ``break``.

## Question 3

Write a JavaScript program that checks numbers from **1 to 100** [ using ``while`` loop ]:

- If a number is divisible by both **3** and **5**, print `"FizzBuzz: <number>"`.
- If a number is divisible by **3** only, print `"Fizz: <number>"`.
- If a number is divisible by **5** only, print `"Buzz: <number>"`.
- **``Skip``** any number that is divisible by **7** using `continue`.
- Keep track of how many numbers have been printed.
- Stop the loop using `break` after **25 outputs** have been printed.
- At the end, print the total number of outputs generated.

## Question 4

Write a JavaScript program that loops through numbers from **1 to 500** [ using ``while`` loop ]:

- **Skip** numbers divisible by **4** using `continue`.
- If a number is divisible by both **6** and **9**, print `"Super Match: <number>"`.
- Else if a number is divisible by **6**, print `"Six Match: <number>"`.
- Else if a number is divisible by **9**, print `"Nine Match: <number>"`.
- Otherwise, print `"Normal: <number>"`.
- Keep a count of `"Super Match"` numbers.
- Stop the loop using `break` after finding **10 Super Matches**.
- Print the total number of iterations performed before stopping.
